Register and update the Director Node

  1. Login to the director node from the Workstation VM:

    ssh stack@director

  2. Register the Director Node

    sudo subscription-manager register

  3. Attach the pool ID and fix the release to rhel8.4.

    sudo subscription-manager attach --pool=FIXME_POOLID
sudo subscription-manager release --set=8.4
    Replace FIXME_POOLID with your actual Red Hat subscription pool ID.

  4. Enable Required Repositories and Modules

    sudo subscription-manager repos --disable=*

sudo subscription-manager repos \
  --enable=rhel-8-for-x86_64-baseos-rpms \
  --enable=rhel-8-for-x86_64-appstream-rpms \
  --enable=rhel-8-for-x86_64-highavailability-rpms \
  --enable=ansible-2.9-for-rhel-8-x86_64-rpms \
  --enable=openstack-16.2-for-rhel-8-x86_64-rpms \
  --enable=fast-datapath-for-rhel-8-x86_64-rpms \
  --enable=advanced-virt-for-rhel-8-x86_64-rpms

sudo dnf module reset container-tools
sudo dnf module enable -y container-tools:3.0

  5. System Update and Reboot

    sudo dnf update -y
sudo reboot
