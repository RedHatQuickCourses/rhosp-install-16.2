Prepare the Director Node

  1. Login to the director node from the workstation VM as student user and switch as root user.

    ssh director
sudo su -

  2. Set the hostname for director node

    echo "director.example.com" > /etc/hostname

  3. Add Stack user and configure it for password less sudo access

    useradd stack
echo "student" | passwd --stdin stack
echo "stack ALL=(ALL) NOPASSWD:ALL" > /etc/sudoers.d/stack

  4. Disable RHEL DVD and run subscription manager cleanup

    mkdir /etc/yum.repos.d/disabled
mv /etc/yum.repos.d/rhel_dvd.repo /etc/yum.repos.d/disabled/
dnf clean all
subscription-manager clean

  5. Expand the filesystem and then reboot

    lsblk
df -h
growpart /dev/vda 3
xfs_growfs /
lsblk
df -h

reboot
    This last step is specific to the ROLE lab environment being offered here.
