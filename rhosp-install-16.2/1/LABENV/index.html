Lab Environment

Instructions to Launch Your Lab on the Red Hat Online Learning (ROLE) Platform

  1. Log in to the ROLE platform.

  2. Search for the catalog entry: Installation of Red Hat OpenStack Platform 16.2 quick Course (or HOL016).

  3. Click on the Launch button for the catalog entry in the search results.

  4. On the Course page, click the Lab Environment tab.

  5. Click on the Create button on the Lab Environment page to launch the lab.

  6. Wait for the lab creation process to complete.

FOR RED HAT ASSOCIATES:

Do NOT refer to the SSH Private Key & Instructions section on the Lab Environment page to access your lab. Use below command to access the lab environment. 
ssh -i /home/ashishks/.ssh/pntae_training_key -J cloud-user@public.ip.of.lab:22022 student@172.25.252.1 -p 53009
SSH key file pntae_training_key is provided in the LMS. Replace the string public.ip.of.lab in the above command with the actual Public Ip on the Lab Environment tab.
